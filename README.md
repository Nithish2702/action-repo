# Simple Calculator

A simple command-line calculator written in Python.

**Repository:** [https://github.com/Nithish2702/action-repo](https://github.com/Nithish2702/action-repo)

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Requirements

- Python 3.6 or higher

## Usage

Run the calculator:

```bash
python calculator.py
```

Follow the prompts to perform calculations.

## Example

```
Enter first number: 10
Enter operation (+, -, *, /): +
Enter second number: 5
Result: 10 + 5 = 15
```
